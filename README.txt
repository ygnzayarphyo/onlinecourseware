This is an online courseware project which Su Wut Yee is the main author in this system. The following are the steps to setup the project...
  
1. Install xampp. This can get from  https://www.apachefriends.org/download.html
3. Add this project under C:/xampp/htdocs/
2. Open xampp application and start Apache and Mysql modules.
3. Click Admin button of Mysql  and import the database under onlinecourseware -> sql -> pcci.sql
4. Then go to 'http://localhost/onlinecourseware' and enjoy this project.
